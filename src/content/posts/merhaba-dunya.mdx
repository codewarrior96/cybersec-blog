## Bir Örnek

Klasik bir nmap taraması nasıl yapılır:

```bash
# Temel servis ve versiyon taraması
nmap -sV -sC -oN scan.txt 10.10.10.1

# Tüm port taraması
nmap -p- --min-rate 5000 10.10.10.1
```

Python ile basit bir port tarayıcı:

```python
import socket

def scan_port(host: str, port: int) -> bool:
    try:
        with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
            s.settimeout(0.5)
            return s.connect_ex((host, port)) == 0
    except Exception:
        return False

for port in range(1, 1025):
    if scan_port("127.0.0.1", port):
        print(f"[+] Port {port} açık")
```
